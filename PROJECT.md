# Projeto

Sistema para uma empresa de móveis planejados.

## Objetivo

Criar um site moderno para geração de orçamentos e apresentação do portfólio da empresa.

## Público

Clientes que desejam cozinhas, banheiros, closets e dormitórios planejados.

## Área de atendimento

Zona Sul de São Paulo.

## Objetivos do sistema

- Mostrar portfólio
- Capturar pedidos de orçamento
- Permitir administração dos projetos
- Melhorar presença digital
- Gerar novos clientes

## Stack

- Next.js
- TypeScript
- Tailwind
- shadcn/ui
- Supabase
- Vercel
